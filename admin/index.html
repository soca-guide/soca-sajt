🔗 Dodeli OWNER (zastarelo)

OWNER nalog kreiraš u Supabase Authentication → Users. Ovde samo vežemo profil + dozvole.

  1. Kreiraj Auth user u Supabase UI (Authentication → Users → Invite / Create user).
  2. Kopiraj user_id (UUID) iz tog unosa.
  3. Nalepi ga ispod i izaberi tenant.
  4. OWNER vidi svoju ulogu pri sledećoj prijavi.
